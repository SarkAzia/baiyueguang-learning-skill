# ChatGPT 安装指南

在 ChatGPT 中使用 **白月光学习法（Baiyueguang Learning Skill）**

---

## 方法一：创建自定义 GPT（推荐）

### 第一步：打开 GPT Builder

进入 ChatGPT：

> Explore GPTs → Create

---

### 第二步：创建 GPT

新建一个 GPT，例如：

- 白月光学习法
- Baiyueguang Learning Skill

---

### 第三步：导入核心 Prompt

打开仓库中的：

```text
SKILL.md
```

复制全部内容。

粘贴到 GPT 的：

```text
Instructions
```

区域。

---

### 第四步：上传知识库

将以下文件上传到 Knowledge：

```text
ONTOLOGY.md
TRANSLATOR.md
examples/
```

推荐全部上传。

作用：

| 文件 | 作用 |
|--------|--------|
| ONTOLOGY.md | 核心概念体系 |
| TRANSLATOR.md | 术语翻译规则 |
| examples/ | 学习案例与示例 |

---

### 第五步：保存 GPT

点击：

```text
Save
```

完成创建。

---

## 开始使用

示例：

```text
请用白月光学习法解释线性无关。
```

```text
请把矩阵翻译成信息关系网语言。
```

```text
请按照 baiyueguang-learning-skill 风格讲解这一章。
```

---

## 推荐模型

效果最佳：

- GPT-5
- GPT-5 Thinking
