# 白月光学习法 Ontology

这个文件定义“学习语言”和“调查语言”的对应关系。

## 核心映射表

| 学习语言 | 调查语言 | 含义 |
|---|---|---|
| 学习对象 | 账号 | 当前要调查的核心对象 |
| 教材 / 论文 / 视频 | 信息源 | 获取信息的地方 |
| 相关概念 | 关注列表 | 它经常和谁一起出现 |
| 重要概念 | 点赞记录 | 高频、关键、有信号价值的信息 |
| 核心知识 | 收藏夹 | 必须保存和反复查看的内容 |
| 前置知识 | 历史记录 | 理解它之前必须看过的东西 |
| 交叉学科 | 共同好友 | 它和其他领域的连接点 |
| 概念关系 | 关系网 | 知识之间的连接结构 |
| 发展过程 | 时间线 | 它如何产生、演化、扩展 |
| 冗余信息 | 重复动态 | 没有提供新信息的内容 |
| 新信息 | 新线索 | 必须单独保留的信息 |

## 核心思想

学习不是直接背定义。

学习是先把一个知识点转换成一个可以调查的对象。

然后围绕它建立：

- 信息源
- 关系网
- 历史记录
- 收藏夹
- 时间线
- 新线索
